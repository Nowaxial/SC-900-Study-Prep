# SC-900 Study Prep

[![Live Site](https://img.shields.io/badge/GitHub_Pages-Live-2ea44f?logo=githubpages)](https://nowaxial.github.io/SC-900-Study-Prep/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE.md)

A comprehensive single-page study hub for the **Microsoft SC-900** exam (*Security, Compliance, and Identity Fundamentals*). Everything runs in one HTML file — no build tools, no dependencies, no server required.

**→ [Live Demo](https://nowaxial.github.io/SC-900-Study-Prep/)**

---

## What's Included

| Section | Content |
|---------|---------|
| **Identity & Access (LP1–2)** | Entra ID, authentication methods, MFA, Conditional Access, PIM, Identity Protection, external identities, hybrid identity |
| **Security Solutions (LP3)** | Defender XDR (MDO, MDE, MDI, MDCA), Defender for Cloud, Microsoft Sentinel, DDoS Protection, Azure Firewall, WAF, NSG, Key Vault, Bastion |
| **Compliance (LP4)** | Microsoft Purview, Compliance Manager, Information Protection, DLP, eDiscovery, Audit, Insider Risk Management, Communication Compliance, Records Management |
| **Security Copilot** | How it works, architecture, integration points |
| **Licensing Guide** | Feature-to-license mapping for Entra ID, M365 E3/E5, Business Premium, Defender, Purview, Sentinel; exam FAQ and real-world scenarios |
| **Flashcards** | 365 searchable flashcards covering all exam domains |

## Features

- **Dark / Light mode** — toggle at the top
- **Full-text search** — instant filtering across all notes and flashcards
- **Tab navigation** — each module in its own tab, just Notes or Notes + Flashcards
- **Flashcards with flip animation** — click to reveal, rate yourself (Got it / Again)
- **12 Microsoft Learn diagrams** — embedded directly from the official modules
- **LocalStorage persistence** — your flashcard progress and theme stay between visits
- **Offline-capable** — everything is self-contained in the HTML file

## How to Use

Open [`index.html`](index.html) in any browser, or visit the [live site](https://nowaxial.github.io/SC-900-Study-Prep/).

- **Navigate** — click any module tab in the sidebar
- **Search** — type in the search bar to filter across all open tabs
- **Flashcards** — open a module's flashcard tab and click a card to flip it
- **Toggle theme** — click the sun/moon icon in the top bar
